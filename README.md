# Smart-Student-Assistant
All-in-one Python GUI app with Calculator, Guess Game, To-Do List, Password Generator

## 🚀 Live Demo
👉   https://smart-student-assistant-bdk4e2j89pbkze3deo5efl.streamlit.app/


---

## 📌 Features

### 🧮 Calculator
- Perform basic operations: Addition, Subtraction, Multiplication, Division
- Handles invalid inputs and divide-by-zero errors

### 🎮 Guess Game
- Guess a random number between 1–10
- Shows correct/wrong result instantly

### 📝 To-Do List
- Add tasks
- View saved tasks
- Stores tasks in a local file (`tasks.txt`)

### 🔐 Password Generator
- Generates secure random passwords
- User can choose password length
